# Layout (布局)

此目录包含负责定义网站整体宏观布局的样式文件。它决定了主要内容区域、页眉、页脚等大型结构元素在页面上的位置和排列方式。

## 文件说明

### `layout.css`

这是布局的核心文件。它主要使用 CSS Grid 和 Flexbox 来构建页面的主要框架，包括：

-   定义主内容区 (`#main-content`) 的网格布局。
-   控制背景包装器 (`#bg-wrapper`) 的样式和过渡效果。
-   管理不同视图（如沉浸式模式）下的布局切换。
-   包含一些高阶的布局容器样式。

#### 自定义说明

此文件主要负责页面的结构布局，其样式（如 `position`, `display`）通常是固定的，不建议修改。该文件中的样式没有通过CSS变量进行参数化，因为它们是网站的基础框架。
